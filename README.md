# Публикуем комиксы в телеграмм

Этот скрипт публикует в телеграмм один случайный комикс xkcd Рэндела Манро.

## Как запустить

Вам потребуется API token вашего бота телеграмм, а так же chat id вашего канала, так же нужно сделать вашего бота администраторм вашего канала.
API token вашего бота присвоить переменной `TELEGRAM_BOT_API_KEY`, chat id вашего канала присвоить переменной `TLGRM_CHAT_ID` в файле .env как в примере ниже:

``` bash
TELEGRAM_BOT_API_KEY=jf85heik58fydjdkfifundk7
TLGRM_CHAT_ID=@channel_id
```

Скрипт запускается командой:
```
python main.py
```

## Необходимое окружение

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```
